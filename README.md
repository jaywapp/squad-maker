# ⚽ 스쿼드 메이커

축구 라인업을 짜고, 공격 패턴을 GIF로 만들어 카톡/단톡방에 공유하는 웹앱입니다.

**🔗 [라이브 데모](https://jaywapp.github.io/squad-maker/)**

---

## 주요 기능

### ⚽ 스쿼드 모드
- **5~11인제 전 지원** — 5v5, 6v6, 7v7, 8v8, 9v9, 10v10, 11v11
- **포메이션 선택** — 모드별 다양한 포메이션 프리셋 (예: 4-3-3, 3-5-2 등)
- **자유 배치** — 선수를 드래그해 원하는 위치로 이동
- **이름 수정** — 더블클릭으로 바로 편집
- **색상 변경** — 우클릭 → 10가지 색상 선택
- **이미지 저장** — PNG로 내보내기 (팀명 + 포메이션 정보 포함)

### 🎯 공격 패턴 모드
- **경로 그리기** — 선수를 클릭-드래그해 이동 경로(화살표) 설정
- **경로 삭제** — 화살표를 우클릭해 개별 삭제
- **패턴 관리** — 여러 패턴을 이름 지정 후 저장, 이전/다음 탐색
- **애니메이션 미리보기** — 부드러운 ease-in-out 재생, 자동 루프
- **GIF 저장**
  - 단일 패턴 GIF
  - 전체 패턴 연속 GIF (패턴 이름 레이블 포함)

## 사용법

### 스쿼드 모드

| 동작 | 기능 |
|------|------|
| 드래그 | 선수 위치 이동 |
| 더블클릭 | 이름 수정 |
| 우클릭 | 색상 변경 / 삭제 |
| `+ 선수 추가` | 선수 추가 (모드별 최대 인원) |
| `↺ 초기화` | 포메이션 기본 배치로 리셋 |
| `📸 이미지 저장` | PNG 다운로드 |

### 공격 패턴 모드

| 동작 | 기능 |
|------|------|
| 선수 클릭-드래그 | 이동 경로 설정 |
| 우클릭 화살표 | 해당 경로 삭제 |
| `▶ 미리보기` | 애니메이션 재생/일시정지 |
| `↺ 경로 초기화` | 현재 패턴 경로 전체 삭제 |
| `🎞 GIF 저장` | 현재 패턴 GIF 다운로드 |
| `📋 전체 패턴 GIF` | 모든 패턴 연속 GIF 다운로드 |

## 기술 스택

- **순수 HTML / CSS / JavaScript** — 빌드 도구 없음, 파일 하나로 실행
- **[html2canvas](https://html2canvas.hertzen.com/)** — PNG 이미지 내보내기
- **[gif.js](https://jnordberg.github.io/gif.js/)** — 브라우저 기반 GIF 인코딩
- **GitHub Pages** — 정적 호스팅

---

## 로컬 실행

별도 설치 없이 `index.html`을 브라우저에서 열면 바로 사용 가능합니다.

```bash
git clone https://github.com/jaywapp/squad-maker.git
cd squad-maker
# index.html을 브라우저로 열기
```

> GIF 내보내기는 CDN에서 gif.js 워커를 불러오므로 인터넷 연결이 필요합니다.
