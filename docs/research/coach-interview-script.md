# 코치 인터뷰 스크립트 (0단계 수요 검증)

- 기준: [수익화 적용 계획서](../2026-07-20-monetization-implementation-plan.md) Task 0.3, Gate A
- 목표: 코치 5~8명에게서 **반복 사용 빈도, 결제 주체, 월 6,900원 가격 반응**을 확인한다.
- 소요: 1인당 20~30분 (전화·화상·대면 무관)

## 진행 원칙

- 기능을 팔지 말고 **현재의 불편을 듣는다**. 유도 질문 금지 — "이 기능 좋죠?" 대신 "지금은 어떻게 하세요?"
- 가격 질문은 마지막에. 먼저 가치를 스스로 말하게 한다.
- 녹음은 사전 동의를 받은 경우에만. 동의 없으면 메모만.
- 인터뷰 요약에는 이름·팀명·연락처를 남기지 않는다(아래 기록 템플릿의 익명 ID 사용).

## 사전 확인 (스크리닝)

1. 어떤 팀을 맡고 계신가요? (연령대·인원제·주당 훈련/경기 횟수 — 개인 식별 정보 제외)
2. 전술이나 라인업을 팀에 전달할 때 주로 어떤 도구를 쓰시나요? (단톡방, 화이트보드, 앱 등)

## 1부 — 현재 워크플로우와 불편 (10분)

3. 최근 경기를 준비하며 라인업·전술을 만든 과정을 처음부터 끝까지 말씀해 주세요.
4. 그 과정에서 가장 시간이 오래 걸리거나 번거로운 지점은 어디였나요?
5. 만든 전술을 다음 경기에서 다시 쓰신 적이 있나요? 어떻게 찾아서 재사용하셨나요?
6. 선수들이 전술을 제대로 확인했는지 어떻게 아시나요? 확인 안 한 선수가 있어 곤란했던 적은?

## 2부 — 후보 기능 반응 (10분)

각 기능을 **한 문장으로만** 설명하고 반응을 듣는다. 순서는 인터뷰마다 섞는다.

7. **클라우드 저장** — "팀과 전술 문서를 계정에 저장해 PC·폰 어디서든 이어서 편집할 수 있습니다."
   - 지금 방식과 비교해 무엇이 달라지나요? 언제 쓰게 될까요?
8. **고급 영상 내보내기** — "패턴 애니메이션을 고화질 영상·PDF로 뽑아 팀에 공유할 수 있습니다."
   - 지금 GIF/이미지로 부족했던 순간이 있었나요?
9. **선수별 브리핑** — "선수마다 자기 위치·지침만 담긴 링크를 보내고, 누가 확인했는지 볼 수 있습니다."
   - 이게 있으면 경기 준비에서 무엇이 바뀌나요?

10. 셋 중 **하나만** 먼저 나온다면 어떤 것을 고르시겠어요? 이유는?

## 3부 — 가격·결제 주체 (5분)

11. 방금 고른 기능이 포함된 Pro가 **월 6,900원(연 59,000원)** 이라면 어떻게 하시겠어요?
    - (즉답을 유도하지 말 것. "고민된다"면 무엇이 걸리는지 묻는다.)
12. 결제한다면 누가 내나요? — 본인 / 팀 회비 / 클럽·기관 예산. 승인 절차가 있나요?
13. 지금까지 팀 운영에 돈을 내 본 도구·서비스가 있나요? 얼마였나요?

## 마무리

14. 출시되면 가장 먼저 알려드릴까요? (연락 동의는 별도로 기록 — 분석 데이터와 분리 보관)
15. 저희가 안 물어본 것 중에 "이건 꼭 알아야 한다" 싶은 게 있을까요?

## 기록 템플릿 (개인정보 제거 요약)

인터뷰 후 24시간 안에 아래 형식으로 `docs/research/interviews/YYYY-MM-DD-C#.md`에 기록한다.
이름·팀명·연락처는 적지 않는다. 연락 동의자는 별도 비공개 목록에만 보관한다.

```markdown
# 인터뷰 C1 (2026-08-01)
- 프로필: 조기축구 11v11, 주 1회 경기, 감독 겸 총무
- 현재 도구: 카카오톡 + 손그림 사진
- 핵심 불편: (본인 표현 그대로 1~2문장)
- 기능 반응: 클라우드 △ / 고급 영상 X / 브리핑 ◎ — 근거 요약
- 1순위 기능: 브리핑 ("전달했는데 안 읽는 게 제일 스트레스")
- 가격 반응: 월 6,900원 — "회비에서 낼 수 있다", 승인: 총무 재량
- 결제 주체: 팀 회비
- Gate A 판정 기여: 구체적 결제 의향 있음 / 없음
```

## Gate A 집계 기준 (계획서 §7.1)

- 인터뷰 5~8건 중 **최소 3명**이 월 6,900원 수준에서 구체적 결제 의향 또는 예산 승인 경로를 설명하면 통과 요건 충족.
- "좋네요, 쓸 것 같아요"는 의향으로 세지 않는다. **누가, 어떤 돈으로, 언제**가 나온 경우만 센다.
